Java程序设计大作业

发表于 2019-05-02 | 更新于 2019-05-03 | 分类于 本科作业 ， java | 阅读次数：

这学期的第一个大作业是java面向对象程序设计，第二个是web，考虑到两个工程可能存在重复性，所以一开始就打定主意，两项作业写同样的东西，区别在于java做成线下管理信息系统；web将java代码重构到web中。

整体代码写完debug完答辩通过之后，就要开始写报告了，决定先在这里理一下整体思路。

大作业要求

  1. 管理信息系统
  2. java swing设计前端界面
  3. 连接数据库实现后端的增删查该

设计思路

由于学校内的跳蚤市场以QQ群的形式出现，内容繁杂，冗余信息多，高峰时发布的物品一扫而过，无法进行分类搜索需要的内容，故打算仿照咸鱼，做出一个校园内部二手交易平台，取名为——“淘客”。
